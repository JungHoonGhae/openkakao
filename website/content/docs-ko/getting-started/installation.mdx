# 설치

## Homebrew

```bash
brew tap JungHoonGhae/openkakao
brew install openkakao-rs
```

업그레이드는 다음처럼 합니다.

```bash
brew update && brew upgrade openkakao-rs
```

## 소스에서 빌드

Rust `>= 1.75`가 필요합니다.

```bash
git clone https://github.com/JungHoonGhae/openkakao.git
cd openkakao/openkakao-rs
cargo install --path .
```

## 확인

```bash
openkakao-rs --version
```

## 셸 자동완성

```bash
# zsh
openkakao-rs completions zsh >> ~/.zfunc/_openkakao-rs

# fish
openkakao-rs completions fish > ~/.config/fish/completions/openkakao-rs.fish

# bash
openkakao-rs completions bash >> ~/.bash_completion
```

## 계속하기 전에

KakaoTalk macOS 앱이 설치되어 있고 이미 로그인되어 있어야 합니다. OpenKakao는 로컬 앱 상태에 의존하며, 새 Kakao 계정 세션을 스스로 bootstrap 하지는 않습니다.
