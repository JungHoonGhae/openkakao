# 빠른 시작

이 경로는 의도적으로 좁게 잡았습니다. 설치하고, 인증하고, 채팅 하나만 확인한 뒤, 신뢰 모델이 자신의 사용 목적에 맞는지 먼저 판단하세요.

## 1. 설치

```bash
brew tap JungHoonGhae/openkakao
brew install openkakao-rs
```

## 2. 인증

KakaoTalk이 실행 중이며 로그인되어 있어야 합니다.

```bash
openkakao-rs login --save
openkakao-rs auth
```

## 3. 채팅 목록 확인

```bash
# 권장: LOCO 기반 채팅 목록
openkakao-rs loco-chats
```

계정 점검과 실제 채팅 접근 중 무엇을 써야 할지 헷갈리면, 자동화 연결 전에 [REST vs LOCO](/ko/docs/getting-started/transport-boundary)를 먼저 읽는 편이 좋습니다.

## 4. 먼저 짧게 읽기

```bash
openkakao-rs loco-read <chat_id> -n 20
```

## 5. 정말 필요할 때만 전송

```bash
openkakao-rs send <chat_id> "Hello from CLI!"
```

> [!NOTE]
> 기본적으로 전송 메시지에는 눈에 보이는 prefix가 붙습니다. 강한 이유가 없다면 유지하는 편이 낫습니다.

## 6. 자동화는 천천히 추가

```bash
openkakao-rs loco-read <chat_id> -n 20 --json | jq '.'
```
