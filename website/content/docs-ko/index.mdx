# OpenKakao

OpenKakao는 개발자, 터미널 친화적 사용자, 자동화 중심 워크플로를 위한 비공식 KakaoTalk macOS CLI입니다. 공식 공개 API가 아니라, 역공학한 KakaoTalk 앱 동작을 통해 개인 채팅에 접근합니다.

> [!WARNING]
> OpenKakao는 강력하지만 무위험 도구는 아닙니다. 실제 계정에 쓰기 전에 [신뢰 모델](/ko/docs/security/trust-model), [데이터와 자격 증명](/ko/docs/security/data-and-credentials), [안전한 사용](/ko/docs/security/safe-usage)을 먼저 읽으세요.

## 가능한 일

| Capability | Why it matters |
|---|---|
| 채팅과 메시지 히스토리 읽기 | 최근 또는 전체 대화를 스크립트와 로컬 도구로 가져오기 |
| 텍스트, 파일, 미디어 전송 | 통제된 아웃바운드 자동화와 운영자 워크플로 구성 |
| 실시간 이벤트 감시 | 메시지 도착 시 스크립트, 알림, 요약 트리거 |
| JSON 내보내기 | KakaoTalk 데이터를 `jq`, `sqlite`, `llm` 등 Unix 도구로 전달 |
| 친구 및 프로필 데이터 확인 | 계정 그래프 기반 개인 워크플로 도구 구성 |

## 문서가 신뢰부터 시작하는 이유

이 프로젝트의 첫 질문은 보통 기능이 아닙니다. 실제 KakaoTalk 계정에 써도 되는지입니다. 문서는 그 질문에 바로 답하도록 구성되어 있습니다.

1. CLI가 사용자 기기에서 무엇을 읽는지
2. 어떤 자격 증명을 로컬에 보관하는지
3. 어떤 네트워크 엔드포인트와 통신하는지
4. 어떤 행동이 계정 리스크를 높이는지
5. 사용 범위를 어떻게 좁고 감사 가능하며 되돌릴 수 있게 유지할지
