## send

LOCO `WRITE` 명령으로 텍스트 메시지를 전송합니다.

```bash
openkakao-rs send <chat_id> "message" [OPTIONS]
```

| Flag | 설명 |
|------|------|
| `-y, --yes` | 확인 프롬프트 건너뜀 |
| `--force` | 오픈채팅으로의 전송 허용 |

`-y`를 쓰려면 unattended send 권한도 같이 허용해야 합니다.

- CLI: `--unattended --allow-non-interactive-send`
- 또는 config: `mode.unattended = true` 와 `send.allow_non_interactive = true`

기본적으로 메시지 앞에는 `🤖 [Sent via openkakao]` prefix가 붙습니다. 끄고 싶다면 전역 `--no-prefix`를 사용하세요.

```bash
openkakao-rs --unattended --allow-non-interactive-send --no-prefix send <chat_id> "raw message" -y
```

## send-file

LOCO `SHIP+POST` 흐름으로 파일(사진, 동영상, 문서)을 전송합니다.

```bash
openkakao-rs send-file <chat_id> <file> [OPTIONS]
```

| Flag | 설명 |
|------|------|
| `-y, --yes` | 확인 프롬프트 건너뜀 |
| `--force` | 오픈채팅으로의 전송 허용 |

텍스트 전송과 마찬가지로 `-y`에는 unattended send 권한이 필요합니다.

미디어 타입은 파일 매직 바이트로 자동 판별합니다.

| Magic Bytes | Type | LOCO Code |
|-------------|------|-----------|
| `FF D8` | JPEG photo | 2 |
| `89 50 4E 47` | PNG photo | 2 |
| `GIF8` | GIF animation | 14 |
| `ftyp` (offset 4) | MP4/MOV video | 3 |
| `1A 45 DF A3` | WebM video | 3 |
| Other | Generic file | 26 |

## send-photo

`send-file`의 별칭입니다. 동작은 동일합니다.

```bash
openkakao-rs --unattended --allow-non-interactive-send send-photo <chat_id> photo.jpg -y
```
