## 사용법

```bash
openkakao-rs [OPTIONS] <COMMAND>
```

## 전역 옵션

| Flag | 설명 |
|------|------|
| `--json` | JSON 출력 |
| `--no-color` | 컬러 출력 비활성화 |
| `--unattended` | non-interactive 실행임을 명시적으로 선언 |
| `--allow-non-interactive-send` | unattended 모드에서 `send -y`와 파일 전송 `-y` 허용 |
| `--allow-watch-side-effects` | unattended 모드에서 watch read receipt, hooks, webhooks 허용 |
| `--no-prefix` | outgoing message 앞의 `🤖 [Sent via openkakao]` prefix 비활성화 |
| `--version` | 버전 출력 |
| `--help` | 도움말 출력 |

`NO_COLOR` 환경 변수도 [no-color.org](https://no-color.org/) 기준으로 존중합니다.

지속적인 정책은 [설정](/ko/docs/getting-started/configuration)에 저장할 수 있습니다.

## 명령

### 인증
| Command | 설명 |
|---------|------|
| `auth` | 토큰 유효성 확인 |
| `login --save` | KakaoTalk에서 자격 증명 추출 |
| `relogin` | `login.json` 기반 토큰 재발급 |
| `renew` | `refresh_token` 기반 토큰 갱신 |

### 채팅 & 메시지 (REST)
| Command | 설명 |
|---------|------|
| `chats` | 채팅방 목록 |
| `read <chat_id>` | 메시지 읽기 |
| `members <chat_id>` | 참여자 목록 |
| `search <chat_id> <query>` | 메시지 검색 |
| `export <chat_id>` | 메시지 내보내기 (JSON/CSV/TXT) |
| `unread` | 읽지 않은 채팅 목록 |

### 채팅 & 메시지 (LOCO)
| Command | 설명 |
|---------|------|
| `loco-chats` | 전체 채팅방 목록 |
| `loco-read <chat_id>` | 메시지 히스토리 읽기 (SYNCMSG) |
| `loco-members <chat_id>` | 참여자 목록 |
| `loco-chatinfo <chat_id>` | 채팅방 상세 정보 |
| `send <chat_id> <msg>` | 텍스트 메시지 전송 |
| `send-file <chat_id> <file>` | 파일 전송 (사진/동영상/문서) |
| `send-photo <chat_id> <file>` | 사진 전송 (`send-file` 별칭) |
| `watch` | 실시간 메시지 감시 |
| `download <chat_id> <log_id>` | 미디어 첨부 다운로드 |

### 친구 & 프로필
| Command | 설명 |
|---------|------|
| `friends` | 친구 목록 |
| `profile <user_id>` | 친구 프로필 |
| `me` | 내 프로필 |
| `settings` | 계정 설정 |
| `profiles` | 멀티 프로필 목록 |

### 유틸리티
| Command | 설명 |
|---------|------|
| `doctor` | 진단 |
| `loco-test` | LOCO 연결 테스트 |
| `scrap <url>` | 링크 미리보기 (OG 태그) |
| `completions <shell>` | 셸 자동완성 |
